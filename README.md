# Python Vector Calculator

## Description

This project is a command-line vector calculator written in Python.  
It allows the user to work with vectors in dimension 2 or dimension 3 and perform common vector operations.

The program asks the user to choose the dimension, enter or modify two vectors, and then select operations from an interactive menu.

## Features

The program can:

- Enter or modify vectors `v` and `w`
- Display a vector
- Add two vectors
- Subtract two vectors
- Multiply a vector by a number
- Calculate the scalar product
- Calculate the norm of a vector
- Normalize a vector
- Calculate the orthogonal projection of one vector onto another
- Calculate the angle between two vectors
- Check if a vector is unitary
- Check if two vectors are collinear
- Check if two vectors are orthogonal
- Calculate the vector product in dimension 3
- Quit the program from the menu

## How It Works

When the program starts, the user chooses whether to work in dimension 2 or dimension 3.

Then, the program displays a menu with several choices.  
The user enters a number corresponding to the operation they want to perform.
